# LibraryProject

This is a Django project for learning Django development.

## Setup

1. Install Django: `pip install django`
2. Navigate to the project directory: `cd Introduction_to_Django`
3. Run the development server: `python manage.py runserver`
4. Visit http://127.0.0.1:8000/ to see the Django welcome page

## Project Structure

- `settings.py`: Configuration for the Django project
- `urls.py`: The URL declarations for the project; a "table of contents" of your Django-powered site
- `manage.py`: A command-line utility that lets you interact with this Django project
- `bookshelf/`: Django app containing the Book model and admin interface

## Project Structure

- `LibraryProject/` - Main Django project settings
