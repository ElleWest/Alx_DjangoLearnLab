# Introduction to Django

This directory contains the Django learning project.

## LibraryProject

The main Django project is located in the `LibraryProject/` subdirectory.

## Setup

1. Navigate to the Introduction_to_Django directory
2. Activate virtual environment: `source /Users/home/alx-fe-reactjs/.venv/bin/activate`
3. Run migrations: `python3 manage.py migrate`
4. Start development server: `python3 manage.py runserver`
5. Visit http://127.0.0.1:8000/

## Apps

- `bookshelf`: Contains the Book model and admin interface
